## 📋 МАТРИЦА ЗАВИСИМОСТЕЙ

```
Онбординг (FSM)
  ↓
  └─ GAME_RULES (созданы)
  └─ GAME_LOG (события)
  └─ PROFILE_GAME (snapshot)
  └─ Attempt #1 (стартован)
       ↓
       └─ Goal decomposition
            ↓
            └─ Quest Architect (GOAL_DECOMPOSER)

Каждый день:
  ENGINE (CRON)
    ├─ Проверить silence (5 дней) → FAILED attempt
    ├─ Проверить STUCK (X дней без денег)
    ├─ Считать stage по STAGE_XP
    ├─ Проверить LEVEL_READY
    └─ Отправить message в Telegram

На любое действие:
  main (router)
    ├─ Прочитать snapshot (6-8 полей)
    ├─ Классифицировать intent (AI)
    ├─ Route (code)
    └─ Target workflow (Quest / Finance / Therapist / etc)
```

**ПРОБЛЕМА:** Если ENGINE неправильно считает stage или attempt, вся система ломается.
