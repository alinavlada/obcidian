📱 iOS Shortcut — Секретутка × Siri

Пошаговая инструкция · Bangkok timezone · n8n webhook

⚡ Прежде чем начать

Убедись что воркфлоу Sekretutka_Siri_Gateway активирован в n8n и ты скопировала его webhook URL. Он выглядит так:

https://ВАШ_N8N_ДОМЕН/webhook/siri-gateway

Если n8n на localhost — для iPhone нужен публичный URL (ngrok, cloudflare tunnel или VPS).

Часть 1 — Создаём Shortcut

1
Открой приложение «Команды» (Shortcuts)
На iPhone → найди приложение Команды (синяя иконка с двумя квадратами). Нажми «+» в правом верхнем углу → «Новая команда».
2
Назови команду «Секретутка»
Нажми на «Переименовать» вверху → введи Секретутка. Это имя Siri будет слышать: «Привет Siri, Секретутка».
3
Добавь действие: «Диктовать текст»
Нажми «Добавить действие» → в поиске напиши Диктовать → выбери «Диктовать текст».
🎙
Диктовать текст
Язык: Русский · Стоп: автоматически

⚠️ Настройки диктовки

Язык: Русский
«Стоп после паузы»: включить (1–2 сек)
«Перефразировать»: выключить

4
Добавь «Получить содержимое URL» (HTTP запрос)
Нажми «+» → найди URL → выбери «Получить содержимое URL». Настрой так:
🌐
Получить содержимое URL
URL: https://ВАШ_N8N_ДОМЕН/webhook/siri-gateway

Параметры запроса

Метод
POST
Заголовки
Content-Type: application/json
Тело запроса
JSON
Ключ: text
Значение: «Продиктованный текст» (вставь переменную из шага 3)

💡 Как вставить переменную

В поле «Значение» нажми на синюю переменную «Продиктованный текст» — это результат предыдущего шага с диктовкой. Если её нет в предложениях — нажми на «Содержимое буфера обмена» и выбери из списка «Продиктованный текст».

Итоговый JSON тела

{ "text": [Продиктованный текст] }
5
Добавь «Получить словарное значение» (парсим ответ)
Нажми «+» → найди Словарь → выбери «Получить значение из словаря».
📖
Получить значение из словаря
Ключ: response · Из: «Содержимое URL» (шаг 4)

⚠️ Важно

Ключ — именно response (строчными). Это поле из JSON-ответа n8n:
{ "response": "Записала! Встреча добавлена на завтра." }

6
Добавь «Произнести текст» (Siri говорит вслух)
Нажми «+» → найди Произнести → выбери «Произнести текст».
🔊
Произнести текст
Текст: «Словарное значение» (результат шага 5)

💡 Настройки голоса

Скорость: 0.5 (комфортно)
Голос: Milena (русский) или Katerina
«Ждать завершения»: включить

Часть 2 — Итоговая схема Shortcut

🎙
1. Диктовать текст
Язык: Русский → сохраняется как «Продиктованный текст»
🌐
2. Получить содержимое URL
POST · JSON · { "text": [Продиктованный текст] } → «Содержимое URL»
📖
3. Получить значение из словаря
Ключ "response" из «Содержимого URL» → «Словарное значение»
🔊
4. Произнести текст
«Словарное значение» · голос Milena · скорость 0.5

Часть 3 — Добавить Siri-фразу

7
Привяжи к Siri
В команде нажми «Настройки» (иконка ⚙️ или кнопка вверху) → «Добавить к Siri» → запиши фразу: «Секретутка» или «Привет Секретутка».

🗣 Как использовать

«Привет Siri, Секретутка» — Siri запустит команду, откроется диктовка. Говори, например:

«Запиши задачу — встреча с клиентом завтра в 14:00»
«Записала воду»
«Потратила 200 бат на такси»
«Какие у меня задачи на сегодня?»

Часть 4 — Проверка и отладка

⚠️ Если Siri не отвечает или ответ пустой

1. Проверь что n8n воркфлоу Sekretutka_Siri_Gateway активен (зелёный)
2. Проверь URL вебхука — он должен быть публичным (не localhost)
3. Запусти воркфлоу вручную через Postman или curl: 

curl -X POST https://ВАШ_N8N_ДОМЕН/webhook/siri-gateway \
  -H "Content-Type: application/json" \
  -d '{"text": "тест"}'

Ожидаемый ответ: {"response": "...", "full_response": "..."}

🔴 Частая проблема: n8n на домашнем сервере

iPhone не достучится до localhost или локальной сети напрямую. Решения:
— Ngrok: бесплатный туннель, команда ngrok http 5678
— Cloudflare Tunnel: стабильнее ngrok, бесплатно
— VPS: задеплоить n8n на DigitalOcean / Hetzner / Contabo

✅ Всё работает — как звучит идеальный диалог

Ты: «Привет Siri, Секретутка»
Siri: [запускает диктовку]
Ты: «Записала воду»
Пауза ~3 сек (n8n обрабатывает)
Siri: «Плюс десять XP. Стакан воды засчитан.»
