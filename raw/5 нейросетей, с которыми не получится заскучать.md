# 5 нейросетей, с которыми не получится заскучать

5 нейросетей, с которыми не получится заскучать

1. [Speechify](https://speechify.com/). Нейронка для озвучивания текста. Отменно работает, разные озвучки, всё правдоподобно.

2. [Promeai](https://www.promeai.com/). Довольно сложный инструмент для рендеринга и работы с фоном. Есть простые аналоги, поэтому для продвинутых пользователей.

3. [Playground AI](https://playgroundai.com/). Отличное решение на тот случай, если вам нужно сгенерировать большое количество картинок, при этом сам промт не сложный.

4. [Microsoft Bing Image Creator](https://www.bing.com/images/create). Генерирует красочные изображения. Работает только с VPN.

5. [Gerwin](https://gerwin.io/ru). Новейший сервис на основе нейронных сетей для генерации потрясающих изображений, рекламы.
